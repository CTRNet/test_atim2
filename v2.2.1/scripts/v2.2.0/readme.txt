ATiM v2.2.0 has two methods for installation:

1 - New installation
For all new installations simply run the following,
Filename: atim_v2.2.0_full_installation.sql

2 - Upgrade
Only one SQL file required to upgrade from v2.1.0A, and only that version. All upgrades are sequential and required. 
Using this script against any earlier versions may cause unpredicible behavior. 
Filename: atim_v2.2.0_upgrade.sql

BE SURE TO BACKUP YOUR DATABASE BEFOREHAND
